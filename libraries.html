    JavaAssist Library

    The JavaAssist Library was thrown together to attempt to make some processes shorter by taking out the long work that needs to be done to accompolish certain tasks. If you would like to check out the java docs for JavaAssist you may do so at javadocs/index.html. If you are trying to find something specific checkout the table down below! With the table you should be able to quickly navigate to the Main Class of each pkg.



    Package Name Brief Description Added Removed
    gui Anything Graphical UI related v1.0 None
    conversion Anything Graphical Conversion related (Temp, currency, etc) v1.0 None
    file Anything File Related v1.0 None
    mathf Extra math functions, more advanced ones to come. v1.0 None
